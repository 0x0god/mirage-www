Mirage has several backends, each of which expose a different level of I/O API depending on what the platform supports. Below is a summary, with the color indicating its implementation status (Red is under design, Orange is partially implemented, and Green means it is working). Keep an eye on the blog for more details about what some of these actually mean, over the next few weeks.
